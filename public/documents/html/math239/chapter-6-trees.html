Trees and Forests

A tree is a connected graph with no cycles
A forest is a graph with no cycles (connectedness not required)

Properties of Trees
...

  • A unique path exists between any two vertices and in a tree
  • Every edge of a tree is a bridge
  • A tree with vertices has edges
    • If a graph is connected with vertices and edges, then is a tree
  • A tree with a least two vertices has at least two leaves

Spanning Trees
...

  • A spanning tree is a spanning subgraph that is also a tree
  • A graph is connected has a spanning tree
  • Adding an edge to a spanning tree produces a cycle; removing a different edge from this cycle produces another spanning tree

...

  • A subgraph of a bipartite graph is also bipartite
  • Graph is bipartite has no cycles of odd length
